Einfach verkettete Liste

Eine Liste, bestehend aus Elementen mit Vorwärtszeigern. Besitzt ein leeres Kopfelement "head" und einen Zeiger "current" (hier innerhalb von 2 '!' gekennzeichnet). Die Knoten haben einen Wert 'data' und einen Verweis auf das nächste Element 'next'.

  • insert(data): Füge Element mit Wert 'data' nach current ein.
  • remove(): Lösche Element nach current.
  • movetofront(): Setze current auf head.
  • next(): Setze current auf nächstes Element in der Liste.
  • end(): Ist current am Ende der Liste?
  • empty(): Ist die Liste leer?
  • read(): Wert des Elements nach current.
  • write(wert): Überschreibe 'data' des Elements nach current.
  • search(wert): Suche das Element rechts von current mit Data = 'wert' und setze current davor.
